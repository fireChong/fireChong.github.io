Posted on 2019-11-10

一、图的概念

图是由顶点集合和顶点间的二元关系集合组成的数据结构，通常可以用G(V, E)来表示。其中顶点集合和边的集合分别用V(G)和E(G)表示。

无向边: 没有方向的边，通常表示为(u,v)，其中u、v分别为这条边的两个顶点，因为与方向无关,所以(u,v)与(v,u)是同一条边。
无向图:所有的边都是无向边的图。
完全图:如果无向图中任何一对顶点之间都有一条边，这种无向图称为完全图

有向边:有方向的边，通常表示为<u, v>，u表示这条边的起点，v表示这条边的终点。
有向图:所有的边都是有向边的图。
有向完全图：如果有向图中任何一对顶点u 和v，都存在<u, v>和<v, u>两条有向边，这种有向图称为有向完全图。

顶点u的度数：与顶点u相关联的边的数目
顶点u的出度:是以u 为起始顶点的有向边的数目
顶点u的入度:是以u 为终点的有向边的数目

路径:在图G(V, E)中，若从顶点vi 出发，沿着一些边经过一些顶点vp1, vp2, …, vpm，到达顶点vj，则称顶点序列(vi, vp1, vp2,…, vpm, vj)为从顶点vi 到顶点vj 的一条路径（或称为通路）。
路径的长度：路径中边的数目通常称为路径的长度。

连通图:从每个顶到到每个其他顶点都存在一条路径的无向图.
强连通图:从每个顶到到每个其他顶点都存在一条路径的有向图.

环:环是一条至少含有一条边且起点和终点相同的路径。
无环图: 一种不包含环的图
二分图:能够将所有节点分为两部分的图，其中图的每条边所连接的两个顶点都分别属于不同的部分。

二、图的常用表示

邻接矩阵:使用一个二维数组，对于每条边(u, v) 置A[u][v] = true; 对于没有的边置A[u][v] = false;
如果边有权，可以置A[u][v] = 权值，对于不存在的边可以置为 +∞ (或者-∞，视情景而定). 所需空间复杂度 O (V^2)。

邻接表: 对每一个顶点，使用一个表存放所有邻接的顶点。空间复杂度 O( V + E)

3.2 深度优先搜索

深度优先搜索算法（英语：Depth-First-Search，DFS）是一种用于遍历或搜索树或图的算法。沿着树的深度遍历树的节点，尽可能深的搜索树的分支。当节点v的所在边都己被探寻过，搜索将回溯到发现节点v的那条边的起始节点。这一过程一直进行到已发现从源节点可达的所有节点为止。如果还存在未被发现的节点，则选择其中一个作为源节点并重复以上过程，整个进程反复进行直到所有节点都被访问为止。

3.2.1 介绍

1.首先将根节点放入stack中。
2.从stack中取出第一个节点，并检验它是否为目标。
如果找到目标，则结束搜寻并回传结果。
否则将它某一个尚未检验过的直接子节点加入stack中。
3.重复步骤2。
4.如果不存在未检测过的直接子节点。
将上一级节点加入stack中。
重复步骤2。
5.重复步骤4。
6.若stack为空，表示整张图都检查过了——亦即图中没有欲搜寻的目标。结束搜寻并回传“找不到目标”。

3.1 广度优先搜索

广度优先搜索算法（英语：Breadth-First-Search，缩写为BFS），又译作宽度优先搜索，或横向优先搜索，是一种图形搜索算法。简单的说，BFS是从根节点开始，沿着树的宽度遍历树的节点。如果所有节点均被访问，则算法中止。

3.1.1 介绍

1.首先将根节点放入队列中。
2.从队列中取出第一个节点，并检验它是否为目标。
如果找到目标，则结束搜索并回传结果。
否则将它所有尚未检验过的直接子节点加入队列中。
3.若队列为空，表示整张图都检查过了——亦即图中没有欲搜索的目标。结束搜索并回传“找不到目标”。
4.重复步骤2。
